# Coaching System

AI-assisted email coaching system for The Launchpad Incubator. Reads participant emails, drafts coaching responses in Wes's voice using GPT-4o, and sends after approval via a Streamlit dashboard.

**Stack:** Supabase + GitHub Actions + OpenAI GPT-4o + Gmail + Streamlit

See [SYSTEM.md](SYSTEM.md) for complete documentation including architecture, database schema, workflows, and configuration.
